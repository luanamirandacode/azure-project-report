📌 Resumo do Projeto
Objetivo: Criar um pipeline real de dados que vai desde a ingestão até a visualização, simulando um cenário corporativo.

Serviços usados:

Azure Data Factory → responsável pela ingestão e orquestração dos dados.

Azure Databricks → utilizado para transformação, limpeza e processamento dos dados em notebooks.

Azure Synapse Analytics → usado como data warehouse para consultas analíticas.

Power BI → ferramenta de visualização para relatórios e dashboards interativos.

🔄 Etapas do Pipeline
Ingestão de dados com Data Factory

Configuração de pipelines para extrair dados de fontes externas (arquivos, bancos, APIs).

Criação de datasets e linked services para conectar ao armazenamento.

Processamento com Databricks

Uso de notebooks para transformar os dados brutos.

Aplicação de limpeza, normalização e enriquecimento dos dados.

Integração com Unity Catalog para governança e segurança.

Armazenamento e análise com Synapse Analytics

Carregamento dos dados tratados no Synapse.

Estruturação em tabelas e modelos analíticos para consultas rápidas.

Visualização com Power BI

Conexão ao Synapse para criar relatórios.

Construção de dashboards interativos que permitem análise de KPIs e métricas de negócio.
